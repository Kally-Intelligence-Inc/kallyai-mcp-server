### 2. Server Description (200-500 characters)
The KallyAI MCP Server enables Claude AI to make phone calls through an AI assistant. Users can schedule restaurant reservations, book medical appointments, make general business inquiries, and handle other phone-based tasks without picking up the phone. Supports English and Spanish with real-time call transcripts and usage tracking.

kallyai-mcp-server

---

### 5. Server Type
Remote MCP Server (supports both Streamable HTTP and stdio transports)

---

### 6. Authentication Method
OAuth2 Bearer Token (local storage)

Authentication tokens are stored locally in ~/.kallyai_token.json and managed automatically by the MCP server. Users authenticate once through the KallyAI web app (https://kallyai.com/app) via Google/Apple Sign-In. The server handles token refresh automatically.

No OAuth configuration needed in the connector - tokens are CLI-managed.

### 10. Usage Examples (Minimum 3)

#### Example 1: Restaurant Reservation

**User Prompt**:
> "Call Bella Italia at +1-415-555-1234 and make a reservation for 4 people tomorrow at 7pm. My name is John Smith."

**Claude Action**:
```
Tool: kallyai_create_call
Parameters:
{
  "task_category": "restaurant",
  "task_description": "Reserve a table for 4 people at 7pm tomorrow",
  "respondent_phone": "+14155551234",
  "business_name": "Bella Italia",
  "user_name": "John Smith",
  "party_size": 4,
  "appointment_date": "2026-01-26",
  "appointment_time": "19:00",
  "timezone": "America/New_York"
}
```

**Result**:
```markdown
# Call Completed

**Status**: success

## Summary
Successfully reserved a table for 4 people at 7:00 PM on January 26, 2026.

## Next Steps
Arrive 10 minutes early. Ask for the Smith reservation.

**Duration**: 2m 15s
```
